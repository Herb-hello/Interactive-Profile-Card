# Interactive-Profile-Card

Interactive Profile Card to practice for my CMPT 272 Intro to Web Dev midterm. I am only using HTML, CSS, and JavaScript to practice the fundementals.

Create an interactive user profile card using HTML, CSS, and JavaScript. The profile card should display a user's picture, name, and a short bio. Upon clicking a "More Info" button, the card should expand to display additional details, such as the user's hobbies, birthdate, and favourite quote.

## Requirements:

## 1. HTML:

- Use semantic HTML elements appropriately.
- The user's name, bio, picture, and additional details must be part of the HTML structure.
- Include a button labelled "More Info" to show additional details.

## 2. CSS:

- Style the profile card so it is visually appealing.
- The profile card should initially have a fixed size, and only the user's picture, name, and bio should be visible.
- Use a CSS transition to smoothly expand the card when the "More Info" button is clicked.
- Make the profile card responsive, ensuring it looks good on all devices.
- Add an animation to the user's picture so that it slightly increases in size when the card expands, and returns to normal when it collapses.

## 3. JavaScript:

- Write a JavaScript function that toggles the visibility of the additional details when the "More Info" button is clicked.
- Implement logic to ensure that the button's text changes to "Less Info" when the additional details are visible and reverts back to "More Info" when hidden.
